Weighted Rating System (WRS)

This frontend is a lightweight, static visualization layer for the Weighted Rating System project. It demonstrates how different ranking algorithms (Bayesian Average, Wilson Score, BTL, Dirichlet–Multinomial) change ordering when vote counts are sparse.
